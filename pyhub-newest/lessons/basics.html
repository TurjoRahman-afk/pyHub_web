File Handling in Python

      
Reading/Writing Text Files

      

        The process of storing data to a file (writing) or retrieving data from a file (reading) so that information can persist outside the program, be shared, or be used later.
      

      
Writing to a Text File

      
with open("team.txt", "w") as file:
    file.write("Hello, Eagle's Ace. These are the members.\n")
    file.write("1. CEO is Turjo\n")
    file.write("2. CTO is Courtney\n")
    file.write("3. CIO is Andrew\n")
    file.write("4. Designers & Document Writers are Keisha and Aya\n")
    file.write("5. Tester is Yasir\n")
      

      
Explanation:

      
    
        
  • "team.txt" → the name of the file we are creating.
    • 
        
  • "w" → write mode → creates the file if it doesn’t exist, or overwrites it if it already exists.
    • 
        
  • with → ensures Python automatically closes the file after writing.
    • 
        
  • file.write() → writes the text into the file.
    • 
      

      
After this block, team.txt will look like:

      
Hello, Eagle's Ace. These are the members.
1. CEO is Turjo
2. CTO is Courtney
3. CIO is Andrew
4. Designers & Document Writers are Keisha and Aya
5. Tester is Yasir
      

      
Reading from the Text File

      
print("Reading the file:")
with open("team.txt", "r") as file:
    content = file.read()
    print("Full content:\n" + content)
   
    file.seek(0)
   
    print("\nLine by line:")
    for i, line in enumerate(file, 1):
        print(f"Line {i}: {line.strip()}")
      

      
Step by Step:

      
    
        
  • "r" → read mode → opens the file to read only.
    • 
        
  • file.read() → reads the entire file content as a single string.
    • 
        
  • file.seek(0) → moves the file pointer back to the beginning.
    • 
        
  • for i, line in enumerate(file, 1): → reads one line at a time with line numbers.
    • 
        
  • line.strip() → removes spaces or newlines at the start/end.
    • 
      

      
Example Output:

      
Line 1: Hello, Eagle's Ace. These are the members.
Line 2: 1. CEO is Turjo
Line 3: 2. CTO is Courtney
...
      

      
Appending to a File

      
with open("team.txt", "a") as file:
    file.write("6. Python Expert: You!\n")
      

      
"a" → append mode → adds new content at the end of the file without deleting existing content.

      
After this, team.txt will now have:

      
6. Python Expert: You!
      

      
Full Code Example

      
def text_file_demo():
    # WRITING to a text file
    with open("team.txt", "w") as file:
        file.write("Hello, Eagle's Ace. These are the members.\n")
        file.write("1. CEO is Turjo\n")
        file.write("2. CTO is Courtney\n")
        file.write("3. CIO is Andrew\n")
        file.write("4. Designers & Document Writers are Keisha and Aya\n")
        file.write("5. Tester is Yasir\n")
   
    # READING from the text file
    print("Reading the file:")
    with open("team.txt", "r") as file:
        # Read entire content at once
        content = file.read()
        print("Full content:\n" + content)
       
        # Reset to beginning of file
        file.seek(0)
       
        # Read line by line
        print("\nLine by line:")
        for i, line in enumerate(file, 1):
            print(f"Line {i}: {line.strip()}")
   
    # APPENDING to the text file
    with open("team.txt", "a") as file:
        file.write("6. Python Expert: You!\n")


# Run the text file demo
text_file_demo()
