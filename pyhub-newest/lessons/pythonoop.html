10. Python OOP

Object-Oriented Programming (OOP) structures code with classes and objects. It makes large programs easier to manage, reuse, and extend.

Classes and Objects

Classes are blueprints, and objects are actual instances built from them.

class Person:
    def __init__(self, name):
        self.name = name

    def coffee(self):
        print(f"{self.name} Buys Coffee.")

person1 = Person("Jackie")
person2 = Person("Maya")

person1.coffee()
person2.coffee()

Output:

Jackie Buys Coffee.
Maya Buys Coffee.

The __str__() Method

Defines how objects are displayed when printed, making them more readable.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"{self.name}({self.age})"

p1 = Person("Keisha", 20)
print(p1)

Output:

Keisha(20)

Creating Methods

Methods are functions inside classes that define object behavior.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def myfunc(self):
        print("Hello my name is " + self.name)

p1 = Person("Keisha", 36)
p1.myfunc()

Output:

Hello my name is Keisha

The self Parameter

self represents the current instance and is used to access attributes and methods.

class Person:
    def __init__(mystuffhere, name, job):
        mystuffhere.name = name
        mystuffhere.job = job

    def myfunc(xyz):
        print("Hello my name is " + xyz.name)
        print("I am a " + xyz.job)

p1 = Person("Keisha", "Student")
p1.myfunc()

Output:

Hello my name is Keisha
I am a Student

Modifying Object Properties

You can change attributes of objects even after creation.

p1 = Person("Keisha", "Student")
p1.job = "Teacher"
print(p1.job)

Output:

Teacher

Inheritance

Inheritance lets one class reuse and extend another class’s features.

class Beverages:
    def __init__(self, temp, drink):
        self.templevel = temp
        self.drinktype = drink

    def printbeverages(self):
        print(self.templevel, self.drinktype)

class Order(Beverages):
    def __init__(self, temp, drink, percentage):
        super().__init__(temp, drink)
        self.sugarlevel = percentage

    def receipt(self):
        print("Hello, your order is", self.templevel, self.drinktype,
              "with", self.sugarlevel, "sugar.")

x = Order("Ice", "Jasmine Tea", "50%")
x.receipt()

Output:

Hello, your order is Ice Jasmine Tea with 50% sugar.

Polymorphism

Polymorphism allows different classes to share the same method name but have different behavior.

class Car:
    def move(self):
        print("Drive!")

class Boat:
    def move(self):
        print("Sail!")

class Plane:
    def move(self):
        print("Fly!")

for vehicle in (Car(), Boat(), Plane()):
    vehicle.move()

Output:

Drive!
Sail!
Fly!
