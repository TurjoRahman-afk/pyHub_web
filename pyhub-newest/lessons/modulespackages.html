Python Modules & Packages

Organizing code, using libraries, and creating your own modules

What is a Python Module?

A Python module is simply a file (.py) that contains functions, classes, or variables. You import it into another file to reuse that code, which helps keep your projects organized.

Importing Python Standard Library Modules

The Python standard library offers more than 200 modules. Example:

import math

result = math.sqrt(16)
print("The value of square root of 16 is", result)
The value of square root of 16 is 4.0

Import with Renaming

import math as m

result = m.sqrt(16)
print("The value of square root of 16 is", result)
The value of square root of 16 is 4.0

Import Specific Functions

from math import sqrt

result = sqrt(16)
print("The value of square root of 16 is", result)
The value of square root of 16 is 4.0

Import All Names

from math import *

result = sqrt(16)
print("The value of square root of 16 is", result)
The value of square root of 16 is 4.0

Python Packages

A package is a collection of modules stored in a folder, usually with an __init__.py file. 

my_package/
    __init__.py
    greetings.py
      
# greetings.py
def say_hello(name):
    return f"Hello, {name}!"
      
from my_package.greetings import say_hello
print(say_hello("Kevin"))
Hello, Kevin!

Writing Your Own Module

# calculator.py
def multiply(a, b):
    return a * b

def division(a, b):
    return a / b
      
# another file
import calculator

print(calculator.multiply(3, 4))
print(calculator.division(20, 5))
